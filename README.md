# bangdream_card_spider
bangdream卡面立绘爬虫

## 特点
- 图片命名条理清晰，易读并方便搜索
- 自动按照乐队名称划分文件夹并储存卡面
- 优先以日语卡面信息命名图片，方便后续更新
## 使用方法
1. 在某个目录下克隆仓库
```
git clone https://github.com/CYDXDianXian/bangdream_card_spider.git
```
2. 安装依赖
```
pip install -r requirements.txt -i https://pypi.douban.com/simple/
```
3. 运行程序
```
py bangdream_card_spider.py
```
